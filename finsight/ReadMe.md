--FinSight--

FinSight is a personal finance project developed using Spring Boot and PostgreSQL. The main purpose of this project is to help users make simple financial decisions based on their income, expenses, savings, EMI, and upcoming bills.

Features-

User Management
ransaction Management
Budget Management
Financial Advisor

The Financial Advisor currently supports:

Lend Money
Buy Decision
Loan Decision

The application checks the user's budget and provides a recommendation, reason, and suggestion.

Technologies Used-

Java
Spring Boot
Spring Data JPA
PostgreSQL
Maven
Postman
Git & GitHub
